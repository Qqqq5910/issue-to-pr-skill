# Issue to PR Skill

A lightweight agent skill that turns GitHub issues into implementation plans, test plans, and pull request descriptions for AI coding agents.

## What it does

- Summarizes the issue
- Identifies likely affected areas
- Creates an implementation plan
- Generates a test checklist
- Writes a clean PR description
- Adds a reviewer checklist

## Best for

- Open-source maintainers
- Indie hackers
- AI coding agent users
- Claude Code / Cursor / Codex workflows

```text

```
