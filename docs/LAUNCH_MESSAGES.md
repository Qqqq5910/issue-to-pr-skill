# Launch Messages

Use these drafts as starting points. Adapt each message to the community before posting.

## Short launch post

```text
I built a small Markdown skill for AI coding agents.

It turns messy GitHub issues into:
- issue summaries
- likely affected areas
- implementation plans
- risk checklists
- test plans
- PR descriptions
- reviewer checklists

The goal is not to auto-code. The goal is to make agent-generated work easier to scope, test, and review.
```

## Technical launch post

```text
I built a small reusable agent skill for GitHub workflows.

The problem: issues are often vague, but AI coding agents tend to start implementation too quickly.

The skill adds a planning layer before code changes begin:
1. restate the issue
2. classify the task
3. identify likely affected areas
4. define an implementation plan
5. add a risk checklist
6. generate a test plan
7. write a PR description
8. prepare a reviewer checklist

It is pure Markdown and has no runtime dependency.
```
